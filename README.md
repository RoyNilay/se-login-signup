## Simple Login and Signup Web page

## Author 
Nilay Roy

## Description 
A basic Login and Signup web application built with HTML and CSS. 
This was created as Homework 1 for ICSI 418Y

## Files included
- login.html - Login page with User ID and Password fields 
- signup.html - Signup page with First Name, Last Name, User ID, and Password Fields 
- styles.css - Provides styling for both pages including layout, form formatting, and button styles
- README.md - Explains the project struct and purpose 

## How to Run 
1. Download all files into the same folder 
2. Open login.html in a web browser
